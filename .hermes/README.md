# Hermes Compatibility Directory

This directory is retained as a migration-time archive for compatibility and
historical tooling only. Its contents are not synchronized after migration.

Active feature briefs, designs, approval markers, and status files live under
`trading/docs/features/`. Development prompts and workflow notes live under
`docs/prompts/` and `docs/development/`.

Do not create new trading policy or feature designs here. This directory is
not a runtime context source, RAG source, rulebook source, or canonical trading
source of truth.
