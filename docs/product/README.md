# Product Docs

Role: compatibility pointer

Canonical Trade_V1 product docs live under:

- `trading/docs/product/`

This root `docs/product/` directory is retained only for compatibility with the
generic Harness layout. It is not trading policy, runtime configuration, LLM
trading context, or rulebook source of truth.

If trading behavior changes, update the canonical product docs, rulebook,
config, or schemas first.
